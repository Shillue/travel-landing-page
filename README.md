# 🌍 Projeto: Adventure Travel Landing Page

Este projeto foi desenvolvido com o objetivo de recriar um design moderno de landing page voltado para viagens e aventuras. A proposta foi transformar um layout visual em uma aplicação real utilizando HTML, CSS e JavaScript, focando em responsividade, interatividade e organização de código.

## 🎯 Objetivo

Praticar a construção de interfaces modernas a partir de um design de referência, aplicando conceitos de layout, responsividade e manipulação do DOM com JavaScript.

## 🖥️ Sobre o Projeto

Adventure Travel é uma landing page inspirada em sites de turismo e exploração. O layout destaca imagens impactantes, tipografia moderna e um carrossel interativo, criando uma experiência visual envolvente para o usuário.

## 📘 Funcionalidades

O projeto conta com os seguintes elementos:

- Header com logo, menu de navegação e área de perfil
- Campo de busca interativo (expansível ao clicar)
- Seção principal com chamada de impacto (hero section)
- Botão de ação “Start Your Adventure”
- Carrossel de imagens dinâmico com navegação por setas
- Contador de slides (01, 02, 03...)
- Layout totalmente responsivo para diferentes telas
- Rodapé com créditos

## 🏷️ Tecnologias e Recursos Utilizados
### HTML
- Estrutura semântica: <header>, <main>, <section>, <nav>, <footer>
- Elementos de conteúdo: <h1>, <p>, <span>, <ul>, <li>, <a>
- Imagens e acessibilidade com atributos alt e aria-label
### CSS
- Flexbox para organização de layout
- Variáveis CSS (:root) para padronização de cores
- Efeitos de hover e transições suaves
- Responsividade com media queries
- Estilização moderna baseada no design original
- JavaScript
- Modularização de código (ES Modules)
- Manipulação do DOM
- Funcionalidades implementadas:
- Toggle do campo de busca
- Criação dinâmica do carrossel
- Controle de navegação (next/prev)
- Atualização do contador de slides

## 🧠 Aprendizados

Durante o desenvolvimento, foram praticados:

- Estruturação de projetos front-end organizados
- Separação de responsabilidades (HTML, CSS, JS)
- Criação de componentes reutilizáveis
- Manipulação dinâmica de elementos com JavaScript
- Implementação de carrossel sem bibliotecas externas
- Aplicação de responsividade para diferentes dispositivos
- Uso de boas práticas de acessibilidade
